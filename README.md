# Goose Grab

A small mobile-friendly match-3-in-a-basket web game, deployable on GitHub Pages.

## How to play

- Tap a tile to send it to the basket.
- When you collect 3 of the same kind in the basket, they clear.
- If the basket fills up (7 slots) you lose.

## Deploy (GitHub Pages)

- Settings -> Pages -> Deploy from a branch -> main / (root)

If this repo is named `goose` under your account, it will be available at:
`https://<username>.github.io/goose/`
